# DUNGEON OF SHADOW
DESENVOLVIDO POR: João Gabriel Vitor Araujo, Laila Saraiva Nascimento e Gabriel dos Anjos Figueiredo;

Uma Sombra antiga que desperta nas profundezas da masmorra. Você é o último Herói da Vila, e deve descer os três andares da masmorra e derrotar o Lorde das Sombras para evitar que o mal vença e consuma o Reino.

Você se movimenta no W, S, D e A, no jogo você precisa pegar uma das 3 armas disponíveis no ferreiro para se preparar para enfrentar a masmorra. Você tem três vidas, e a cada vez que você leva dano para um monstro ou para o espinho, perde uma das vidas, você ataca os monstros apertando o botão O. A cada andar você precisa pegar as chaves, interagindo tanto com as chaves, quanto para abrir a porta ou quando interagir com NPCs, para assim abrir as portas responsavéis por você descer um andar na masmorra pelas escadas. Para voltar ao Menu, aperte Q. No Menu terá três opções, 1 para jogar, 2 para Tutorial e 3 para sair do jogo. Seu objetivo é derrotar o Boss Final, assim zerando o jogo. Caso suas vidas zerem, você leva Game Over e volta desde o início.

Cada símbolo tem seu significado, os seus significados são:
^ < > v:  Jogador (direção em que olha);
*: Parede (intransponivel);
#: Espinho (mata ao tocar!);
k: Caixa (destrutível com ataque);
O: Botao (interaja com I);
D: Porta fechada;
@: Chave (interaja com I para coletar);
=: Porta aberta;
L: Escada (próxima fase);
X: Monstro Tipo 1 (movimento aleatório);
Y: Monstro Tipo 2 (persegue você);
Z: Boss Final;
N: NPC (fale com I);
M: Entrada da masmorra;

Sobre o uso da IA, usamos principalmente para ajudar com o jogo no geral e para entendermos o código do jogo.
