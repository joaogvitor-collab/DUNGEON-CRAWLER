# DUNGEON OF SHADOW
DESENVOLVIDO POR: João Gabriel Vitor Araujo, Laila Saraiva Nascimento e Gabriel dos Anjos Figueiredo;

Uma Sombra antiga que desperta nas profundezas da masmorra. Você é o último Herói da Vila, e deve descer os três andares da masmorra e derrotar o Lorde das Sombras para evitar que o mal vença e consuma o Reino.
